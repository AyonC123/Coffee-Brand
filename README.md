# Coffee-Brand

## Project setup

1.

```
npm i -D parcel
yarn i -D parcel
```

2.

```
npm i -D sass
yarn i -D sass
```

## Run Project

3.

```
to run both
npm run dev
yarn run dev
```

2.

```
to run parcel
npm run parcel
yarn run parcel
```

3.

```
to run scss
npm run scss
yarn run scss
```

## Build Project

```
npm run build
yarn run build
```
